Companion reference to [`hobart-plan.md`](./hobart-plan.md). The core document picks one recommended course per live goal; this file contains the full menu (every reasonable low-writing option), the RateMyProfessors / forum sentiment per course, and alternate paths if Jack wants to swap any of the recommendations.

## How HWS aspirational goals work

Jack has 5 goal-course slots (Goal #1–#5 in the shared core) plus 1–2 free-elective slots over Y1–Y3. The physics major auto-covers **Quantitative Reasoning** and **Scientific Inquiry** (confirm both major courses carry those PeopleSoft attributes), so the four "live" goals Jack has to hit are:

- **Artistic Process (AP)**
- **Social Inequality (SI)**
- **Cultural Differences (CD)**
- **Ethical Judgment (EJ)**

**HWS rule:** 1 "Substantially"-tagged course = goal closed; 2 "Partially"-tagged courses = goal closed. A minimum of 5 distinct courses must be applied to goals in total (the 2 physics-major goal contributions count toward this).

All entries below were cross-checked against the [HWS Educational Goals "Courses by Goal" database](https://campus.hws.edu/EducationalGoals/Home/CoursesByGoal). **Goal attributes are re-reviewed annually** — Jack must verify the attribute is still attached in PeopleSoft at registration before locking in a pick.

## Difficulty legend (used in every table below)

| Bucket        | RMP difficulty score | Practical meaning                                                            |
| ------------- | -------------------- | ---------------------------------------------------------------------------- |
| **Very easy** | ≤ 2.0                | Participation grade OR explicitly called "easy A" by multiple RMP reviewers  |
| **Easy**      | 2.0–2.9              | Most students earn As with modest effort                                      |
| **Moderate**  | 3.0–3.5              | Standard college difficulty; outcome depends on student effort               |
| **Hard**      | 3.6–4.5              | Tough grader, heavy workload, or both                                        |
| **Very hard** | > 4.5                | Multiple RMP reviewers flag as unusually demanding                            |
| **Unknown**   | no RMP data          | Assume moderate; ask Prof. Spector / FYS adviser / teammates before locking  |

**Range notation** (e.g., "Easy–Hard (2.3–4.4)"): the course is taught by multiple instructors whose RMP difficulty scores span the range. The "target instructor" column names the one to pick for Easy-A outcomes; the others should be avoided if minimum difficulty is the priority.

**Forum caveat:** HWS has a minimal Reddit footprint (r/hws is near-dormant, r/HobartWilliamSmith is private). The most useful public signal is [RateMyProfessors.com/school/412](https://www.ratemyprofessors.com/school/412), ~300 HWS professor profiles. RMP skews toward extreme opinions and small-N ratings for a small college (~1,800 UG) can be noisy — the same course with a different professor can swing from Easy-A to brutal, so **always check the specific Fall-offering professor on RMP before registering**.

## Double-count gems (2 goals closed in 1 course)

These are the highest-leverage picks — each collapses a goal-slot by half. Two gems exist for Jack's live-goal set (confirmed against the Courses-by-Goal list):

- **ANTH 110 Intro Cultural Anthropology** = Substantial SI + Substantial CD. Closes 2 of Jack's 4 live goals in 1 course. 2–3 short papers. RMP difficulty 2.0 (Easy) with Maiale.
- **DAN 110 Dances of the African Diaspora** = Substantial AP + Substantial CD. Closes 2 of Jack's 4 live goals in 1 course. 0 writing (studio-based). No RMP data but studio-graded.

Taking both = 3 of 4 live goals closed in 2 courses (AP, SI, CD — the overlap on CD makes 1 of them redundant for CD-coverage, but both still count as distinct goal-applied courses for the 5-distinct rule). Jack then needs only a single EJ course + optional AP "depth" course to close everything — a 3-goal-course total for all 4 live goals, the minimum possible.
